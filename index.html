Fitness & Dietary Application; A Level coursework:


Overview:

For my A level coursework I was tasked with creating a solution to a common task or problem I deal with on a daily basis.

Therefore I decided to create a fitness & dietary application that enables me to scan/search for a food item and nutritional information will be returned on request. Calorie intake will also be tracked and a tailored goal will be set for the user.

The point of the project was to demonstrate that we could create a working solution to a problem - not produce a piece of ground-breaking of software.

Technical aspects:

My application was coded for Android using Android Studio. The front-end was created using XML, and the back-end with Java.

My applciation linked to a mySQL database that contained a huge number of food items and their corresponding nutritional information.

All technical aspects mentioned above were self-taught and coded over the course of ~4 months.
